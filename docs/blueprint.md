# **App Name**: Utility King AI

## Core Features:

- Conversational Hero: An interactive hero section that uses Gemini to understand user needs and guide them through the service.
- AI-Powered Comparison Tool: Integrate with Gemini API (via `/api/compare`) to fetch and normalize real-time pricing data, then use Gemini to provide intelligent comparisons based on user input, displayed in a carousel or card layout.
- Service Cards: Display available services using cards with icons and hover-lift animations, including Energy Tariff Switch, Broadband Plan Match, Mobile Data Savings, Smart Meter Insight, and Bill Breakdown Explainer. Adapt to a vertical stack on mobile.
- Value Proposition Grid: Showcase key value propositions (100% impartial, Backed by data, UKi AI-powered help, Live rates, No hidden fees, Free to use) in a grid layout, complete with animated number counters to highlight statistics like Users Helped and Avg Savings.
- Animated CTA Banners: Implement full-width parallax CTA banners with scroll-triggered fade-in effects using `framer-motion`. Types include a 'Quote Prompt' and 'Chat Prompt.'
- Persistent AI Assistant: Integrate a floating chat head icon powered by the Gemini API, accessible via text and voice input. The AI assistant serves as a tool that will elicit soft email lead prompts during conversations.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) for a sense of trust and reliability.
- Background color: Light gray (#F0F4F8), a desaturated version of the primary, for a clean and professional look.
- Accent color: Teal (#38B2AC) for CTAs and interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for headlines, offering a techy, modern look; Body font: 'Inter' (sans-serif) for longer text, ensuring readability.
- Code font: 'Source Code Pro' (monospace) for displaying any code snippets within the application.
- Use clear and modern icons from a set like Phosphor or Remix Icon, related to utilities, savings, and AI assistance.
- Subtle parallax scrolling effects for banners, hover-lift animations for service cards, and scroll-triggered fade-ins for content sections to enhance user engagement.
