What counts as broadband setup costs?
Activation fee, engineer installation, new line installation, full fibre install, router delivery, hardware upgrades. Not all providers itemise clearly – always review the breakdown.

May need engineer visit to install fibre to property. Typical £0–£60, but many providers offer free install on 18–24 month contracts.

New line installation costs
If property has no active line, is new build, or never had broadband: £50–£100+. Some providers include free in promotions – check.
Router charges explained

Most providers include router free, but watch for delivery fees, premium upgrades, early termination hardware fees. May need to return router after contract.

Moving home & setup costs

Some charge relocation fees, others waive during transfer. New fibre installation may apply. Compare switching vs transferring – switching can offer better overall value.

Hidden costs to watch
✔️ VAT included? (business plans often excl.) ✔️ Mid‑contract price rises (CPI +3.9%) ✔️ Setup waived only for online orders? ✔️ Free install requires min. contract length?

When paying setup fees is worth it

If monthly price is significantly lower, contract flexibility needed, shorter commitment wanted, or full fibre install improves reliability – paying setup can be sensible. Short‑term thinking may cost more long‑term.

FAQs
